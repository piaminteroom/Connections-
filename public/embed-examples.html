ConnectSphere Widget Integration Guide

Embed professional networking discovery on your website in minutes

üöÄ Quick Start

Add the ConnectSphere widget to your website in 3 simple steps:

Step 1: Include the Script

<script src="https://your-domain.com/embed.js"></script>

Step 2: Add the Widget Container

<div data-connectsphere></div>

Step 3: Customize (Optional)

<div data-connectsphere data-cs-title="Find Your Network" data-cs-primary-color="#3b82f6" data-cs-compact-mode="true"> </div>

üìù Example 1: HTML Data Attributes (Recommended)

The easiest way to embed the widget with custom styling:

üîß Example 2: JavaScript API

For more control and dynamic configuration:

<div id="my-networking-widget"></div> <script src="https://your-domain.com/embed.js"></script> <script> // Initialize widget with custom configuration const widget = new ConnectSphereWidget('my-networking-widget', { title: 'Corporate Networking Tool', primaryColor: '#059669', accentColor: '#047857', compactMode: false, showPoweredBy: false, // Event callbacks onConnectionsFound: function(connections) { console.log('Found connections:', connections); // Custom analytics tracking gtag('event', 'connections_found', { 'connections_count': connections.length }); }, onError: function(error) { console.error('Widget error:', error); // Custom error handling } }); // Update configuration dynamically widget.updateConfig({ primaryColor: '#dc2626', title: 'Updated Title' }); </script>

üì± Example 3: Compact Mode

Perfect for sidebars, mobile views, or space-constrained areas:

<div data-connectsphere data-cs-compact-mode="true" data-cs-title="Quick Connect" data-cs-max-width="300px" data-cs-show-logo="false"> </div>

üì° Event Handling

Listen to widget events for custom integrations:

// Listen for custom events document.addEventListener('connectsphere:connections', function(event) { const connections = event.detail.connections; console.log('New connections found:', connections); // Example: Send to your analytics analytics.track('Connections Found', { count: connections.length, targetCompany: connections[0]?.company }); }); document.addEventListener('connectsphere:error', function(event) { const error = event.detail.error; console.error('ConnectSphere error:', error); // Example: Show custom error message showNotification('Unable to find connections. Please try again.', 'error'); });
